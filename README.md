# 프로젝트 제안서_마음나래

# **프로젝트 이름: 마음나래**

나래는 날개를 뜻하는 순우리말로

마음이 불편한 상태일 때 해당 프로그램을 사용하여 날개를 단 듯이 자유롭고 편안한 마음을 갖게 된다는 뜻

<aside>
💡

### 프로젝트 설명:

개선하고 싶은 마음상태에 대한 신체증상, 감정, 행동을 프로그램에 입력하여, 자신의 상태를 인지하고 극복할 수 있는 방법을 상담의 개입 양식(행동, 인지, 정서)의 체계를 통해 알려주는 프로그램

</aside>

### 프로그램을 만드는 이유:

- 평소 AI를 통해 고민을 이야기 하거나 불쾌한 정서를 개선했음
- 그러나 기존 AI의 목적은 사용자가 계속 AI를 사용하도록 유도하는 데에 있음
- AI의 목적을 사용자와 치료에 두어
- 해당 프로그램을 통해 사용자가 실제로 행동하고 자신을 가꾸도록 함

### 예상 사용자:

- [ ]  첫번째 사용자는 “나”
- [ ]  AI에게 심리상담을 자주 하는데, 계속 대화만 하게되고 실질적인 해결에 있어서 불편함을 느끼는 사람
- [ ]  정서적 에너지가 자주 소진되어 평소에 마음챙김이 필요한 사람

### 결과물 형태: CLI(프롬프트 입력 형태)

### 사용 예상 python 개념:

- 변수, 조건문, 함수, 리스트, 딕셔너리, 튜플, 집합 등
